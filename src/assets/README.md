# Asset Guide

Naming convention: kebab-case, semantic role first.

## Brand
- `brand-logo.svg` / `brand-logo.png` - full EduFlow logo.
- `brand-mark.svg` / `brand-mark.png` - icon-only mark.

## App Backgrounds
- `bg-panel.png` - main dark AI panel background used in app hero/login panels.
- `bg-mountains.png` - light decorative background, reserved for calm full-width sections.

## Empty And Status States
- `state-error.png` - error/failure states.
- `state-no-lessons.png` - no lessons yet.
- `state-no-students.png` - no students/people state.
- `state-no-tasks.png` - no quiz/tasks yet.
- `state-success.png` - success/completed action.

## Workflow Illustrations
- `hero-voice-recording.png` - lesson recording and audio workflow.
- `hero-invite-students.png` - inviting/adding students.
- `hero-ai-class-analysis.png` - AI analytics or class analysis hero.
- `hero-morning-report.png` - reports/summary headers.
- `hero-key-visual.png` - broad product key visual.

## Metrics
- `metric-completed-lessons.png` - completed lessons/progress.
- `metric-student-activity.png` - student activity.
- `metric-learning-gaps.png` - weak areas / areas to improve.
- `metric-ai-recommendations.png` - AI recommendations.
